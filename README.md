# Fastify Todo API

API with Fastify JS with simple own authentication

## Getting Started

1. clone this repository
2. npm install
3. setup the .env
4. node fastify.js

## Built with

* [Fastify](http://fastify.io/) - The web framework used
* [MySQL](https://mysql.com/) - Database
